Functionality

This project is a website for a pharmacy. It has a login system for the employees to access the website and check the stock of the pharmacy. The website also has a user management area where admins can control basic user privileges. Also, this website includes a billing section to manage receipts and create new receipts/billings for customers. The system also includes a reporting area where admins can check the sales with some basic accounting reports.
